      1.7 数値計算とグラフ化¶

      SIR 数理モデル（感染症数理モデル）¶

      $$ \frac{\mathrm{d}S}{\mathrm{d}t}(t) = -\beta S(t)I(t) $$
      $$ \frac{\mathrm{d}I}{\mathrm{d}t}(t) = \beta S(t)I(t) - \gamma I(t) $$
      $$ \frac{\mathrm{d}R}{\mathrm{d}t}(t) = \gamma I(t) $$

      総人口 = 未感染者数 + 感染者数 + 回復者（免疫保持者）の数¶

      S(t) = 時間 t における未感染者数¶

      I(t) = 時間 t における感染者数¶

      R(t) = 時間 t における回復者（免疫保持者）数¶

      計算手順¶

      第0ステップ
      初期値の設定
      総人口 N : 1000万人
      初期感染者 I(0)=100人
      初期未感染者数 S(0)=10000000-100
      初期回復者 R(0)=0

      感染率 β=m×p/N
      m: 1日１人あたり平均接触人数
      （ここではm=10人に設定）
      p: 接触毎に感染が生じる1日あたりの確率
      （ここではp=0.02に設定）
      感染率β=10×0.02/N
      回復率 γ=1/回復日数
      （ここでは感染者の回復日数を14日に設定)
      回復率γ=1/14=0.071…

      第1ステップ
      dS(ある時間の未感染者数の変化量) = -(感染率) (初期未感染者数) (初期感染者数)
      dI(ある時間の感染者数の変化量) = (感染率) (初期未感染者数) (初期感染者数) - (回復率) (初期感染者数)
      dR(ある時間の回復者数の変化量) = (回復率)       (初期感染者数)

      S（ある時間後の未感染者数） = S0（初期未感染者数） + dS(ある時間の未感染者数の変化量)
      I（ある時間後の感染者数） = I0（初期感染者数） + dI（ある時間の感染者数の変化量）
      R（ある時間後の回復者） = R0（初期回復者数） + dR(ある時間の回復者数の変化量)

      第2ステップ-第N回（日数）ステップ

      dS(ある時間の未感染者数の変化量) = -(感染率) (現在の未感染者数) (現在の感染者数)
      dI(ある時間の感染者数の変化量) = (感染率) (現在の未感染者数) (現在の感染者数) - (回復率) (現在の感染者数)
      dR(ある時間の回復者数の変化量) = (回復率)       (現在の感染者数)

      S（ある時間後の未感染者数） = tempS（現在の未感染者数） + dS(ある時間の未感染者数の変化量)
      I（ある時間後の感染者数） = tempI（現在の感染者数） + dI（ある時間の感染者数の変化量）
      R（ある時間後の回復者） = tempR（現在の回復者数） + dR(ある時間の回復者数の変化量)

      第2ステップ-第N回（日数）ステップを繰り返し計算することで感染者数の日時変化をシミュレーションできる．

      In [66]:
      import numpy as np

#第0ステップ
N = 1000 * 10**4  #総人口 N
I0 = 100  #初期感染者
S0 = N - I0  #初期未感染者数
R0 = 0  #初期回復者

beta = 10 * (0.02/N)  #感染率 
gamma = 1/14  #回復率


#第1ステップ
#dS/dt =　-beta*S*I
#dI/dt = beta*S*I-gamma*I
#dR/dt = gamma*I

dS = -(beta) * S0 * I0  #ある時間の未感染者数の変化量
dI = (beta) * S0 * I0 - (gamma) * I0  #ある時間の感染者数の変化量
dR = (gamma) * I0  #ある時間の回復者数の変化量


#第2ステップ

#現在の未感染者，感染者，回復者の値を一時的に格納（代入）する．
tempS = S0 + dS  #現在の未感染者数
tempI = I0 + dI  #現在の感染者数
tempR = R0 + dR  #現在の回復者数


#現在の未感染者，感染者，回復者の値を時系列として保存するために，リストを作成する．
S = list()
I = list()
R = list()

#現在の未感染者，感染者，回復者の値をリストに格納する．
S = np.append(S, tempS)
I = np.append(I, tempI)
R = np.append(R, tempR)

#未感染者，感染者，回復者の値を350回（350日）繰り返し計算する．
for i in range(350):
    
    dS = - (beta) * tempS * tempI  #ある時間の未感染者数の変化量
    tempS = tempS + dS  #現在の未感染者数
    S = np.append(S, tempS)  #現在の未感染者数を格納

    dI = (beta) * tempS * tempI - (gamma) * tempI  #ある時間の感染者数の変化量
    tempI = tempI + dI  #現在の感染者数
    I = np.append(I, tempI)  #現在の感染者数を格納
    
    dR = (gamma) * tempI  #ある時間の回復者数の変化量
    tempR = tempR + dR  #現在の回復者数
    R = np.append(R, tempR)  #現在の回復者数を格納

      3.4 回帰とは¶

      重回帰分析¶

      説明変数 $x_1,x_2,x_3,⋯,x_m$
      予測値（回帰モデル） $ŷ$
      回帰係数 $w_1,w_2,w_3,⋯,w_m$

      回帰モデル $ŷ =w_1x_1+w_2x_2+⋯+w_mx_m$

      アイリスデータセットの場合
      説明変数は4個
      $x_1$ : がくの長さ(Sepal Length)
      $x_2$ :がくの幅 (Sepal Width)
      $x_3$ :花びらの長さ (Petal Length)
      $x_4$ :花びらの幅 (Petal Width)

      目的変数 $y$ 150データ
      0: setosa（セトサ）種 50データ
      1: versicolor（バージカラー）種 50データ
      2: virginica（バージニカ）種 50データ

      $y$ と $x_1$, $x_2$, $x_3$, $x_4$ を用いて回帰係数$w_1,w_2,w_3,w_4$を求められれば，回帰モデル$ŷ$を導ける
      *解き方の詳細は教科書等で確認してください

      4.2 回帰の場合：実測値と予測値による評価（数値による評価）¶

      MAE (Mean Absolute Error:平均絶対誤差)¶

      $$ MAE = \frac{1}{N} \sum_{i=1}^{N}|y_i-\hat{y_i}| $$

      $MAE$は実測値$y$と予測値$\hat{y_i}$の差の平均（誤差平均）を表す．

      MaxAE (Maximum Absolute Error:最大絶対誤差)¶

      $$ MaxAE(c)=max|x_i−c| $$

      R (Pearsonの相関係数)¶

      $$ r = \frac{\sum_{i=1}^{N}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^{N}(x_i-\bar{x})^2} \sqrt{\sum_{i=1}^{N}(y_i-\bar{y})^2}} $$

      2つの数値データ間$(x_1, y_1),(x_2, y_2),...,(x_i, y_i)$の線型性の強さを示す（相関係数）．

      R2 (R-squared:決定係数)¶

      $$ r^2 = 1 - \frac{\sum_{i=1}^{N}(y_i-\hat{y_i})^2}{\sum_{i=1}^{N}(y_i-\bar{y_i})^2} $$

      MSE (Mean Square Error:平均二乗誤差)¶

      $$ MSE = \frac{1}{N}\sum_{i=1}^{N}(y_i-\hat{y_i})^2 $$

      RMSE (Root Mean Square Error:二乗平均平方根誤差)¶
