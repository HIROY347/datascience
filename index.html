      1.7 数値計算とグラフ化¶
      SIR 数理モデル（感染症数理モデル）¶
      

      

      

      

      

      

      
$$
\frac{\mathrm{d}S}{\mathrm{d}t}(t) = -\beta S(t)I(t)
$$

      

      

      

      

      

      
$$
\frac{\mathrm{d}I}{\mathrm{d}t}(t) = \beta S(t)I(t) - \gamma I(t)
$$

      

      

      

      

      

      
$$
\frac{\mathrm{d}R}{\mathrm{d}t}(t) = \gamma I(t)
$$

      

      

      

      

      

      

      総人口 = 未感染者数 + 感染者数 + 回復者（免疫保持者）の数¶
      S(t) = 時間 t における未感染者数¶
      I(t) = 時間 t における感染者数¶
      R(t) = 時間 t における回復者（免疫保持者）数¶

      import numpy as np

#0
N = 1000 * 10**4  # N
I0 = 100  #
S0 = N - I0  #
R0 = 0  #

beta = 10 * (0.02/N)  # 
gamma = 1/14  #


#1
#dS/dt =-beta*S*I
#dI/dt = beta*S*I-gamma*I
#dR/dt = gamma*I

dS = -(beta) * S0 * I0  #
dI = (beta) * S0 * I0 - (gamma) * I0  #
dR = (gamma) * I0  #


#2

#
tempS = S0 + dS  #
tempI = I0 + dI  #
tempR = R0 + dR  #


#

I = list()
R = list()

      4.2 回帰の場合：実測値と予測値による評価（数値による評価）¶
      

      

      

      

      

      

      

      MAE (Mean Absolute Error:平均絶対誤差)¶
      

      

      

      

      

      

      
$$
MAE = \frac{1}{N} \sum_{i=1}^{N}|y_i-\hat{y_i}|
$$

      

      

      

      

      

      

      $MAE$$y$$\hat{y_i}$
      


      

      

      

      

      

      

      MaxAE (Maximum Absolute Error:最大絶対誤差)¶
      

      

      

      

      

      

      
$$
MaxAE(c)=max|x_ic|
$$

      

      

      

      

      

      

      R (Pearsonの相関係数)¶
      

      

      

      

      

      

      
$$
r = \frac{\sum_{i=1}^{N}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^{N}(x_i-\bar{x})^2} \sqrt{\sum_{i=1}^{N}(y_i-\bar{y})^2}}
$$

      

      

      

      

      

      

      2$(x_1, y_1),(x_2, y_2),...,(x_i, y_i)$ 
      


      

      

      

      

      

      

      R2 (R-squared:決定係数)¶
      

      

      

      

      

      

      
$$
r^2 = 1 - \frac{\sum_{i=1}^{N}(y_i-\hat{y_i})^2}{\sum_{i=1}^{N}(y_i-\bar{y_i})^2}
$$

      

      

      

      

      

      

      MSE (Mean Square Error:平均二乗誤差)¶
      

      

      

      

      

      

      
$$
MSE = \frac{1}{N}\sum_{i=1}^{N}(y_i-\hat{y_i})^2
$$

      

      

      

      

      

      

      RMSE (Root Mean Square Error:二乗平均平方根誤差)¶
      

      

      

      

      

      

      
$$
MSE = \sqrt{\frac{1}{N} \sum_{i=1}^{N}(y_i-\hat{y_i})^2}
$$
